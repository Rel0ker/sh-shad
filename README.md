# Изменения в расписании (школа)

Локальное приложение: завуч вводит строки изменений, сохраняет день и скачивает **Excel**, **PDF**, **PNG** и **XLSX приказа** об оплате часов.

## Требования

- Python **3.11+** — только для разработки и сборки exe
- На школьных ПК: **один файл** `schedule_changes.exe`, Python не нужен
- **Первый запуск exe:** интернет (~150 МБ) — программа сама скачает Chromium для PDF/PNG в профиль Windows (`%LOCALAPPDATA%\ScheduleChanges\`). Дальше интернет не обязателен
- **Excel** работает сразу, даже если Chromium не скачался

## Запуск из исходников

```bash
cd "путь/к/проекту"
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

При старте Chromium проверяется и при необходимости скачивается автоматически.

## Сборка для Windows (один файл на флешку)

На компьютере с Python запустите **`build_windows.bat`**.

Результат: **`dist/schedule_changes.exe`** — только этот файл отправляете в школу.

| Что происходит в школе | |
|------------------------|---|
| Первый запуск | Окно консоли: загрузка Chromium (нужен интернет) |
| Следующие запуски | Проверка за секунду, сразу открывается сайт |
| `app.db` | Создаётся **рядом с exe** (данные школы) |
| Chromium | В `%LOCALAPPDATA%\ScheduleChanges\` (не рядом с exe) |

Сборка только **на Windows**. Если загрузка Chromium не удалась (прокси, антивирус) — Excel всё равно работает; PDF/PNG заработают после успешного перезапуска с интернетом.

## Справочники

Раздел **«Справочники»**: учителя, классы (смена 1 или 2), предметы. Импорт CSV/XLSX/XML.

## Экспорт

- **Excel — учителя / ученики** — без браузера
- **PDF / PNG** — через Chromium (автоустановка)
- **Приказ ЗП** — шаблон `data/templates/zameny_zp_template.xlsx`

## Структура

- `app.py` — Flask, автозапуск браузера, проверка Chromium при старте
- `core/playwright_bundle.py` — загрузка и путь к браузеру
- `core/db.py` — SQLite
- `build_windows.bat` / `build.spec` — сборка одного exe

## Лицензия

Используйте в школе свободно.
